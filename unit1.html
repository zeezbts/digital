LOGIC GATES

1. Basic Gates

Basic gates form the core foundation of all digital systems.

  • AND Gate: Outputs 1 only when all inputs are 1. It acts as logical multiplication.
  • OR Gate: Outputs 1 when at least one input is 1. It acts as logical addition.
  • NOT Gate: Inverts the input signal completely. A 1 becomes 0, and a 0 becomes 1.

2. Universal Gates

Universal gates can mimic any other logic gate configuration on their own.

  • NAND Gate: An AND gate followed by a NOT gate. Outputs 0 only when all inputs are 1.
  • NOR Gate: An OR gate followed by a NOT gate. Outputs 1 only when all inputs are 0.

3. Derived Gates

Derived gates are complex combinations of basic gates used for arithmetic operations.

  • XOR (Exclusive-OR) Gate: Outputs 1 only when inputs are different from each other.
  • XNOR (Exclusive-NOR) Gate: Outputs 1 only when inputs are identical to each other.
