Detailed Guide on Parity Generators

A parity generator is a combinational logic circuit designed to add an extra bit—called a parity bit—to a binary data word before it is transmitted. Its primary purpose is to enable error detection at the destination.

Even vs. Odd Parity

1. Even Parity Generator

The goal is to make the total number of 1s in the complete message (data bits + parity bit) even.

  • If the data has an even number of 1s: The data is already compliant. The generator outputs a 0.
  • If the data has an odd number of 1s: The generator outputs a 1 to make the overall count even.

4-bit even parity generator

2. Odd Parity Generator

The goal is to make the total number of 1s in the complete message odd.

  • If the data has an even number of 1s: The generator outputs a 1 to make the overall count odd.
  • If the data has an odd number of 1s: The data is already compliant. The generator outputs a 0.

Detailed Guide on Parity Checkers

A parity checker is a combinational logic circuit deployed at the receiver end of a digital communication system. Its primary role is to inspect incoming data blocks to verify whether data corruption occurred during transmission.

Even vs. Odd Parity

1. Even Parity Checker

This circuit verifies if the total count of 1s in the entire received block is even.

  • Valid State: Total count of 1s is even → Checker outputs 0 (No error detected).
  • Corrupted State: Total count of 1s is odd → Checker outputs 1 (Error detected).

4-bit even parity checker

2. Odd Parity Checker

This circuit verifies if the total count of 1s in the entire received block is odd.

  • Valid State: Total count of 1s is odd → Checker outputs 0 (No error detected).
  • Corrupted State: Total count of 1s is even → Checker outputs 1 (Error detected).
